
Ker: Knowledge Semantic Data Base

Usage:
  ker [command]

Available Commands:
  --------------- DB Control ------------------

  create [db name]   Create a new DB
  mv [db name]       Move between DBs
  set                Change data base presets
  ls                 List data bases available
  
  --------------- DB Access -------------------

  add  [files]       Add files to a DB
  madd               Add input content DB
  use                Chat with LLM
  usem               Chat with embedding

  --------------- API -------------------------

  run      Run the DB as an API
                  * Chat with embeddings
                  * Chat with LLM
                  * Add content
                  * Modify settings
                  * running config on presets
  stop     Stop a Database API Server

For more details check https://github.com/OmarSaldanna/ker-knowledge-sdb
